# Archive

Здесь хранится всё, что выводится из активной части проекта, но не должно быть потеряно:

- старые документы;
- архивы `progress.md`;
- результаты чисток;
- устаревшие артефакты и промежуточные файлы;
- любые данные, которые раньше могли бы быть удалены.

Базовое правило: из проекта ничего не удаляем без крайней необходимости. Если что-то больше не нужно в рабочем контуре, это переносится в `archive/`.
