# kanatka2

Автоматический отбор лучших фотографий с горнолыжной канатной дороги.

Камера снимает проезжающих на подъёмнике лыжников. Программа группирует кадры в серии по времени, отбрасывает пустые кресла, выбирает лучшее фото в каждой серии и собирает печатные листы для типографии.

---

## Возможности

- **Автоматическая группировка** — кадры объединяются в серии по временны́м меткам файлов.
- **Двухэтапная детекция** — MediaPipe face detection + Haar upper body cascade (fallback для людей в шлемах/очках).
- **Скоринг по 3 критериям** — наличие человека (40), резкость (35), освещение (25).
- **Выбор лучшего кадра** — из каждой серии автоматически выбирается фото с максимальным score.
- **Сборка печатных листов** — сетка 2×4, готовые JPEG для печати.
- **Веб-интерфейс** (`localhost:8787`) — основной UI:
  - Просмотр серий с карточками, бейджами и оценками.
  - Временно́й браузер соседних серий (±3) для поиска нужных кадров.
  - Batch-выбор фото и отправка на печать.
  - Страница настроек инженера: 20+ параметров с ползунками и подсказками.
- **ZIP-экспорт** — фильтрация по дате (сегодня / неделя / диапазон / всё).
- **Автосинхронизация** — листы и rescued-фото копируются в сетевую папку.
- **Debug overlay** — 3-колоночная таблица score поверх фото (Человек, Резкость, Свет).

## Архитектура

```
Камера → INBOX/ → [Анализ + Скоринг] → selected/ + rejected/ → [Листы] → sheets/
                                                                         ↓
                                                            Сетевая папка (LAN)
                                                                         ↓
                                                              Типография (печать)
```

**Два компьютера в одной LAN:**

| Компонент | Роль | Требования |
|-----------|------|------------|
| Основная программа | Автономная обработка, веб-интерфейс администратора | Windows, CPU-only |
| Программа-приёмник | Приём листов, показ оператору типографии | Лёгкая, i3/8GB |

Основная программа работает без оператора — запустил и забыл. Типография получает готовые листы через сетевую папку.

## Быстрый старт

```bash
# Установка зависимостей
pip install opencv-python pillow mediapipe

# Запуск
python src/main.py
```

Программа откроет tkinter-launcher. Выберите папку с фото (по умолчанию `INBOX/`) и нажмите «Обработать». Веб-интерфейс доступен по адресу `http://localhost:8787`.

## Конфигурация

Все параметры в `src/config.json`. Основные секции:

| Секция | Что настраивает |
|--------|-----------------|
| `scoring_weights` | Веса 3 компонентов скоринга |
| `thresholds` | Пороги детекции и качества |
| `haar_cascade` | Параметры Haar upper body (scaleFactor, minNeighbors, minSize) |
| `series_detection` | Группировка серий (max_gap, cooldown) |
| `sheet` | Параметры печатных листов (сетка, размер, качество) |
| `network` | Путь сетевой папки, автосинхронизация |

Все параметры доступны через веб-интерфейс на странице «Настройки».
