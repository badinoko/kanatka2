# PhotoSelector — Канатка

Автоматический отбор лучших фотографий с горнолыжной канатной дороги.

Одна программа. Один компьютер. Запустил — работает.

## Что делает

Камера снимает лыжников на кресельном подъёмнике. Программа:

1. Мониторит папку с входящими фото.
2. Группирует кадры в серии по времени создания файла.
3. В каждой серии выбирает лучший кадр.
4. Отбрасывает пустые кресла.
5. Собирает печатные листы (сетка 2x4).
6. Печатает автоматически или сохраняет в тестовом режиме.

## Установка

Запустите `PhotoSelector_Setup.exe` и следуйте инструкциям.

После установки:
- Программа в меню Пуск: **PhotoSelector**.
- Рабочие папки создаются автоматически в `workdir/` рядом с программой.

## Тестирование

1. Запустите **PhotoSelector**.
2. Распакуйте тестовый архив в папку `INBOX\` рядом с программой.
3. В интерфейсе нажмите **Запустить** (мониторинг входящих фото).
4. Запустите `simulate_camera.bat`, чтобы подать тестовые фото в `workdir\incoming\`.
5. Через несколько секунд на главной странице появятся обработанные серии.

Подробнее: `docs/user_testing_guide.md`.

## Интерфейс

Веб-интерфейс на `localhost:8787`:

- **Главная** — список серий с карточками, бейджами, оценками (1-5 звёзд).
- **Листы** — собранные печатные листы, preview, ручная печать, увеличение по клику.
- **Настройки** — 35+ параметров: пути, grouping серии, `delta_score`, автопечать, принтер (пароль по умолчанию: `1234`).
  Для серий важны параметры `Макс. разрыв внутри серии` и `Ожидание тишины перед разбором`.

## Детекция

Двухэтапная:
- **MediaPipe Face Landmarker** — основной детектор лиц.
- **Haar upper body cascade** — fallback для людей в шлемах, очках, масках.

## Скоринг

3-слойная модель: occupancy → quality gate → ranking.

7 компонентов ranking: читаемость лица (30), ракурс (15), резкость лица (20), освещение лица (15), кол-во читаемых (10), качество кадра (8), улыбка (2). Сумма = 100.

## Стек

- Python 3.11+, Windows, CPU-only
- OpenCV, Pillow, MediaPipe, watchdog
- pywebview (WebView2) + встроенный HTTP-сервер
- Без внешних веб-фреймворков

## Структура

```
src/                    Исходный код
  app.py                Точка входа (pywebview)
  series_browser.py     Веб-интерфейс (~1500 строк)
  watcher.py            Мониторинг входящей папки
  analyzer.py           Анализ фото
  scorer.py             Скоринг (7 компонентов)
  selector.py           Выбор лучшего кадра
  sheet_composer.py     Сборка печатных листов
  config.json           Все настройки
tools/
  camera_simulator.py   Симулятор камеры (для тестов)
tests/                  52 теста (unittest)
build/                  PyInstaller + Inno Setup
```

## Рабочие папки

```
workdir/
  incoming/     Входящие фото от камеры (сюда кидать фото)
  selected/     Лучшие фото (по одному на серию)
  sheets/       Печатные листы
  discarded/    Пустые кресла
  rejected/     Проигравшие при отборе
  archive/      Обработанные фото
  ambiguous/    Спорные серии (ручная проверка)
  logs/         Логи
```

Все пути настраиваются через веб-интерфейс (Настройки → Рабочие папки).
Для локального тестирования по умолчанию также используется папка `INBOX/`.
