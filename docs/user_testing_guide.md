# PhotoSelector v2 — Как тестировать

## Установка

1. Запустите `PhotoSelector_Setup_v2.exe`.
2. Установщик создаст все нужные папки автоматически, включая знакомую папку `INBOX`.
3. По умолчанию программа ставится в `C:\Users\<имя>\AppData\Local\Programs\PhotoSelector`.

## Где лежат рабочие папки

После установки в папке программы:

```
PhotoSelector/
  PhotoSelector.exe          Сама программа
  simulate_camera.bat        Тестовая подача фото из INBOX
  process_folder.bat         Обработка папки с фото
  INBOX/                     СЮДА МОЖНО РАСПАКОВАТЬ INBOX.zip
  workdir/
    incoming/                Сюда программа получает входящие фото
    selected/                Лучшие фото (результат)
    sheets/                  Печатные листы (результат)
    discarded/               Пустые кресла
    rejected/                Проигравшие кадры
    logs/                    Логи
```

## Рекомендуемый способ для заказчика: INBOX + simulate_camera.bat

1. Распакуйте архив с вашими тестовыми фото в папку `INBOX\`.
2. Запустите **PhotoSelector**.
3. На главной странице нажмите **Запустить** у мониторинга.
4. Дважды щёлкните `simulate_camera.bat`.
5. Если батник спросит источник, укажите папку `INBOX\`.
6. Подождите, пока серии появятся на главной странице.

Этот способ ближе всего к реальной работе программы: фотографии не обрабатываются “одним куском”, а подаются в `workdir\incoming\` постепенно.

## Способ 2: Кинуть фото прямо в incoming

1. Запустите **PhotoSelector** (из меню Пуск или с рабочего стола).
2. В верхней панели нажмите **Запустить** — загорится зелёный индикатор.
3. Скопируйте ваши тестовые фото в папку `workdir\incoming\`.
4. Подождите 10-15 секунд.
5. На главной странице появятся обработанные серии.

**Важно:** если скопировать большую пачку одновременно, мониторинг может получить её как один большой поток. Для более реалистичной проверки лучше использовать `simulate_camera.bat`.

## Способ 3: Обработать готовую папку целиком

Если у вас есть папка с фото и вы хотите обработать всё разом:

1. Откройте папку программы в проводнике.
2. Перетащите вашу папку с фото на `process_folder.bat`.
   Или откройте командную строку и выполните:
   ```
   cd "C:\Users\<имя>\AppData\Local\Programs\PhotoSelector"
   process_folder.bat "D:\мои_фото"
   ```
3. Запустите **PhotoSelector** и посмотрите результат.

## Что вы увидите

### Главная страница — серии

Каждая карточка = одна скамейка канатки (группа кадров, снятых подряд).

- **selected** — лучший кадр выбран автоматически.
- **ambiguous** — программа не уверена, проверьте глазами.
- **discarded** — пустое кресло, людей нет.
- **Звёзды** — оценка качества (1-5).

Нажмите на карточку → увидите все кадры серии и почему выбран именно этот.

### Листы

Вкладка **Листы** — готовые печатные листы (сетка 2x4 фото).

Клик по превью листа открывает его в полном размере. Так удобно проверять компоновку и score overlay.

По умолчанию включён **тестовый режим** — листы формируются, но НЕ печатаются.

### Настройки

Вкладка **Настройки** → пароль `1234`.

Здесь можно менять:
- Как поток фото режется на серии по времени.
- Порог спорного выбора (`delta_score`) и включение manual review.
- Чувствительность распознавания людей.
- Пути к рабочим папкам (если нужно изменить расположение).

Главные настройки серии:
- **Макс. разрыв внутри серии** — если между соседними кадрами по времени создания разрыв не больше этого значения, они остаются в одной серии.
- **Ожидание тишины перед разбором** — сколько программа ждёт после последнего нового файла, прежде чем начать разбирать накопившуюся очередь.

## На что обращать внимание

1. **Серии** — правильно ли группируются? Не сливаются ли соседние кресла и не режется ли одна серия на две?
2. **Выбор** — действительно ли выбирается лучший кадр?
3. **Пустые кресла** — корректно ли отбрасываются?
4. **Листы** — устраивает ли качество и компоновка?

Записывайте замечания — мы подстроим параметры.
