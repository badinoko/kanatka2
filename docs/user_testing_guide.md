# PhotoSelector v2 — Инструкция для заказчика

## Что это такое

PhotoSelector автоматически отбирает лучшие фотографии с горнолыжной канатки.

Камера снимает проезжающих лыжников. Программа группирует кадры по сериям, в каждой серии выбирает лучшее фото, собирает печатные листы и отправляет в типографию.

Сейчас вы работаете с тестовой версией: программа выбирает фото, но печать не происходит (настроен тестовый режим).

---

## Как начать тест

1. Запустите **PhotoSelector** (значок на рабочем столе).
2. Откроется интерфейс программы.
3. Убедитесь, что в папке `INBOX` (рядом с программой) есть фотографии.
4. Нажмите кнопку **▶ Запустить** в навигационной панели.
5. Программа запустит мониторинг и автоматически начнёт подавать фото из `INBOX`.
6. Подождите 30-60 секунд — на экране появятся обработанные серии.

**Важно:** При нажатии «Запустить» появится уведомление «ТЕСТОВЫЙ РЕЖИМ: симулятор подаёт фото из INBOX». Это нормально — в реальной работе фото поступают от камеры автоматически, без симулятора.

## Что вы увидите

### Страница «Серии»

Каждая карточка = один проезд кресла канатки (группа кадров, снятых подряд).

- **Выбрано** — лучший кадр выбран автоматически. Звёзды — оценка качества (1-5).
- **Спорное** — программа не уверена, рекомендует проверить глазами.
- **Пусто** — пустое кресло, людей нет.

Нажмите на карточку — увидите все кадры серии и почему выбран именно этот.

### Кнопка «Рядом»

Показывает соседние серии по времени. Удобно, если нужно выбрать другой кадр вручную.

### Страница «Листы»

Готовые печатные листы (сетка 2×4 фото). Клик по превью открывает лист в полном размере.

По умолчанию включён **тестовый режим** — листы формируются, но не печатаются.

| Элемент | Что делает |
|---|---|
| **Серии** | Главная — список всех серий |

| **Настройки** | Параметры программы (пароль: `1234`) |
| **Обновить** | Перезагрузить страницу |

| **Очистка** | Удалить рабочие данные и начать заново |
| **📖 Инструкция** | Эта инструкция |
| **Индикатор диска** | Свободное место (зелёный / жёлтый / красный) |
| **▶ Запустить** | Запустить мониторинг + подачу тестовых фото |

## Настройки

Вкладка **Настройки** → пароль `1234`.

Ключевые параметры:
- **Макс. разрыв внутри серии** — если пауза между кадрами меньше этого значения, они попадают в одну серию.
- **Ожидание тишины перед разбором** — сколько ждать после последнего файла перед обработкой.
- **Мин. разница score** — если разница между лучшим и вторым кадром мала, серия помечается как спорная.

Пути к рабочим папкам тоже можно менять через настройки.

---

## На что обращать внимание

1. **Серии** — правильно ли группируются кресла? Не сливаются ли два соседних в одну серию?
2. **Выбор кадра** — действительно ли выбирается лучшее фото из серии?
3. **Пустые кресла** — корректно ли они отбрасываются?
4. **Листы** — устраивает ли компоновка (2×4)?

Записывайте замечания — параметры можно откалибровать.

## Рабочие папки

После обработки файлы распределяются по папкам:
- `workdir\selected\` — лучшие фото (по одному на серию)
- `workdir\sheets\` — печатные листы
- `workdir\rejected\` — остальные кадры серий
- `workdir\discarded\` — пустые кресла
- `workdir\logs\` — логи и отчёты
